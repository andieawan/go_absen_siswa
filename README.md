# Aplikasi Absensi Sekolah

## Struktur
- `index.html` - Entry point
- `style.css` - Styling
- `script.js` - Logika frontend
- Backend: Google Apps Script (code.gs) + Google Spreadsheet

## Deploy
1. Push ke GitHub
2. Update versi di index.html (?v=YYYYMMDD)
3. GitHub Pages otomatis deploy
